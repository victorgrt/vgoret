Makefile

"Quitte à en faire un autant qu'il soit jolie"

Un makefile est un fichier utilisé par la commande "make" pour automatiser la compilation de programmes en C ou en C++. Il spécifie les cibles à construire, ainsi que les dépendances et les commandes nécessaires pour construire chaque cible. L'utilisation d'un makefile présente de nombreux avantages pour les développeurs. Tout d'abord, cela permet d'automatiser la compilation de programmes, ce qui peut être particulièrement utile lorsque le code source est volumineux ou que de nombreux fichiers sont impliqués dans le projet. En utilisant un makefile, vous n'avez pas à retaper manuellement les commandes de compilation chaque fois que vous modifiez le code source. Pour créer un makefile, il suffit de créer un fichier texte nommé "makefile" (ou "Makefile" si votre système d'exploitation est sensible à la casse) et de commencer à écrire les règles de compilation. Chaque règle commence par le nom de la cible, suivi d'un signe ":", puis des dépendances séparées par des espaces. Ensuite, vous pouvez écrire les commandes de compilation nécessaires pour construire la cible en utilisant une tabulation au début de chaque ligne. Voici un exemple de makefile simple qui compile un programme "hello.c" : hello: hello.c gcc -o hello hello.c Ce makefile contient une seule règle qui indique que la cible "hello" dépend du fichier "hello.c" et que la commande "gcc -o hello hello.c" doit être utilisée pour la construire. Pour utiliser ce makefile, vous pouvez simplement exécuter la commande "make" dans votre terminal, et le programme "hello" sera automatiquement compilé.

            # ! ARGUMENTS ! #
            
            NAME	=	libft.a
            CCFLAGS	=	-Wall -Werror -Wextra
            CC		=	gcc
            
            # * SRC * #
            
            SRC = ft_atoi.c ft_bzero.c ft_calloc.c ft_isalnum.c ft_isalpha.c ft_isascii.c \
                ft_isdigit.c ft_isprint.c ft_itoa.c ft_memchr.c ft_memcmp.c ft_memcpy.c	\
                ft_memmove.c ft_memset.c ft_putchar_fd.c ft_putendl_fd.c ft_putnbr_fd.c	\
                ft_putstr_fd.c ft_split.c ft_strchr.c ft_strdup.c ft_striteri.c ft_strjoin.c \
                ft_strlcat.c ft_strlcpy.c ft_strlen.c ft_strmapi.c ft_strncmp.c ft_strnstr.c \
                ft_strrchr.c ft_strtrim.c ft_substr.c ft_tolower.c ft_toupper.c \
            
            OBJ = ${SRC:.c=.o}
            
            # ? RULES ? #
            
            all: ${NAME} 
            
            ${NAME} : ${OBJ}
                ${CC} ${OBJ} -o ${NAME} (...)
